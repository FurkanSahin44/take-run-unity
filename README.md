
# üçé Take&Run ‚Äì Unity Gameplay Prototype

A small Unity gameplay prototype built to practice basic game loops, movement mechanics, and enemy behavior.

The player must pick up an apple and reach the target while avoiding enemies that patrol the area.  
This project was developed to experiment with Unity fundamentals and improve C# scripting skills.

---

## Features
- Player movement (C#)
- Enemy patrol & chase behavior
- Collectable item system (apple)
- Win/Lose states + restart logic
- Basic UI elements (score / message)
- Scene & prefab organization
